# Security Policy

## Reporting a vulnerability

If you've found a security issue in PlanetPulse infrastructure or any of our products, please email:

**hayyan@planetpulse.life** (subject: SECURITY)

Please include:

- A description of the issue
- Steps to reproduce, if known
- Affected component
- Your contact info if you'd like a follow-up

## Response

We aim to acknowledge within **2 business days** and provide a remediation timeline within **5 business days** for confirmed issues.

## Please do not

- Open public GitHub issues for security vulnerabilities
- Test for vulnerabilities against production tenants other than your own
- Run automated scanners against our production hosts without coordinating with us first

## Recognition

We're a small team and don't currently run a paid bug bounty, but we'll publicly credit reporters (with permission) once issues are resolved.
